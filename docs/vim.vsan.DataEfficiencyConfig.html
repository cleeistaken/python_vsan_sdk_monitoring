Data Object Description

The data efficiency configuration of a vSAN cluster.

Properties

Name Type Description
compressionEnabled*xsd:boolean

Is data compression enabled on the cluster? If data compression enabled, new created all-flash disk group will do data compression. If disabled, new created disk group will not do data compression.
dedupEnabledxsd:boolean

Is data deduplication enabled on the cluster? If data deduplication enabled, new created all-flash disk group will do data deduplication. If disabled, new created disk group will not do data deduplication.
