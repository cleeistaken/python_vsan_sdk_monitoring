Data Object Description

An IpConfigEx is an extension of vim.vsan.host.IpConfig, it contains a pair of IPv4 multicast addresses and a pair of IPv6 multicast addresses, used by vSAN service. For vSAN there is one such IpConfigEx per "virtual network" as represented by vim.vsan.host.ConfigInfo.NetworkInfo.PortConfigEx. On vSAN vmknic with both IPv4 and IPv6 addresses, vSAN attempts communication over both IP versions. Setting the IPv6 related configuration values here only if we want to change them from their defaults.
