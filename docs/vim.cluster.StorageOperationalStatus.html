Since

Data Object Description

Additional information on the effects of backend resources and operations on the storage object.

Properties

Name Type Description
healthy*xsd:boolean

Whether the object is currently affected by the failure of backend storage resources.
operationETA*xsd:dateTime

Estimated completion time of a backend operation affecting the object. If set, then "transitional" will be true.
operationProgress*xsd:long

Percent progress of a backend operation affecting the object. If set, then "transitional" will be true.
transitional*xsd:boolean

Whether an object is undergoing a backend operation that may affect its performance. This may be a rebalancing the resources of a healthy object or recovery tasks for an unhealthy object.
